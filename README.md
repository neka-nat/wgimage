# wgimage

[![crates.io](https://img.shields.io/crates/v/wgimage.svg)](https://crates.io/crates/wgimage)

WebGPU Image Processing Library.

## Getting started

```sh
git clone https://github.com/neka-nat/wgimage.git
cargo run --release --example grayscale
```
